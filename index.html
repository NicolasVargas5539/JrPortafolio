Mapa en Angular con Mapbox

Esta es una aplicación de Angular la cual por medio de MapBox y una conexión con sockets realizamos la generación de un mapa con diversos marcadores que pueden ser visualizados desde diferentes navegadores al mismo tiempo, es decir podemos realizar cambios en los marcadores y estos se visualizan en tiempo real.

Para realizar este proyecto, los conocimientos requeridos son: Conocimientos en Angular y el manejo de Sockets, tener conocimientos en TypeScript y conocimientos HTML, CSS para el diseño que vera el usuario final.
